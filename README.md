## 📖 О проекте

Проект содержит реализацию двух задач по работе с **динамическими структурами данных** (однонаправленные списки, деревья) в рамках индивидуального задания по программированию. Обе задачи решены с использованием собственных структур данных без применения специализированных библиотек.
