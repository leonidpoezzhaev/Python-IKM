## 📖 О проекте

Проект содержит реализацию двух задач по работе с **динамическими структурами данных** (однонаправленные списки, деревья) в рамках индивидуального задания по программированию. Обе задачи решены с использованием собственных структур данных без применения специализированных библиотек.


### 🎯 Задача №8 — Приведение подобных в многочлене

Программа представляет многочлен в виде **однонаправленного линейного списка**, где каждый узел содержит коэффициент и степень. Выполняется полный цикл обработки:

- Парсинг строки с многочленом
- Приведение подобных членов
- Сортировка по убыванию степени
- Вывод результата в удобочитаемом формате

**Пример работы:**
```python
Ввод:  3x^2 - 2x + 5 + 4x^2 - x
Вывод: 7x^2 - 3x + 5
```

### 🎯 Задача №20 — Построение дерева выражений и упрощение

Программа строит бинарное дерево выражений из формулы в инфиксной записи, а затем выполняет упрощение по правилам:

(f1 * f2 * f3) → ((f1 + f2) * f3)

(f1 * f2 * f1 * f3) → (f1 * (f2 + f3))


**Возможности:**

-Поддержка операций: +, -, *, /

-Унарный минус

-Целые числа и однобуквенные переменные

-Вложенность скобок до 10 уровней

Ввод:  (a*b*c) + (d*e*d*f)
Вывод: ((a+b)*c) + (d*(e+f))
